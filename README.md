# event-management-dApp
The WEB3 version of an Event Management app like Eventbrite where we can host events and attend events.

### **Deployed Link:**
https://event-management-d-app.vercel.app/

### **Tech Stacks used:**
- ReactJs
- Solidity 
- NextJs
- The Graph
- Infura
- Web3.Storage
- Ethers.js
